## ‚öôÔ∏è CUSTOMIZATION GUIDE:

### Change Brand Name:
Find and replace "BARETTA" in all HTML files

### Change Colors:
In `style.css`, find the `:root` section:
```css
:root {
    --bg-primary: #070B1D;  /* Your background color */
    --accent: #7B5CFF;       /* Your accent color */
    --text: #EAEAFF;         /* Your text color */
    --glow: #B3A7FF;         /* Your glow color */
    --silver: #C0C0D8;       /* Your secondary color */
}
```

### Change Products:
In `shop.html`, find the `.product-card` sections and update:
- Image URLs
- Product names
- Categories
- Prices
- Badges

### Change Images:
Replace Unsplash URLs with your own:
```html
<img src="YOUR-IMAGE-URL" alt="Description">
```

### Disable 3D (for performance):
In `script.js`, comment out the animate() function call

## üéØ CART FUNCTIONALITY:

**How it works:**
1. Click any product in shop page
2. Cart counter increases in navigation
3. Item stored in browser localStorage
4. Navigate to cart page
5. See all items with quantities
6. Update quantities or remove items
7. See live total calculation
8. Click "Proceed to Checkout" (demo alert)

**Data Persistence:**
- Cart data saved in localStorage
- Persists across page refreshes
- Survives browser closes
- Clears only when manually emptied

## üêõ TROUBLESHOOTING:

**3D not showing?**
- Check internet connection (Three.js loads from CDN)
- Open browser console (F12) for errors
- Try Chrome browser

**Images not loading?**
- Check internet connection (images from Unsplash CDN)
- Replace with local images if needed

**Cart not working?**
- Check if JavaScript is enabled
- Clear browser cache
- Check browser console for errors

**Page looks broken?**
- Make sure all files are in same folder
- Check if CSS file loaded
- Refresh the page (Ctrl+F5)
